Intallation in Localhost

Step 1:

  1. Download and unzip the Latest WordPress package
  2. Drop the files in your root folder.
  3. Enter your root path in the browser

Step 2:

  1. When you entered correct root path in the browser, wordpress redirects you to the installation page.
  2. You have to create one DB for this particular wordpress build. Goto http://localhost/phpmyadmin/
  3. Add a new database in dashboard page and add an user account details in privileges tab.
  4. Come again to the browser and Enter All the DB details. Click Install.

Step 3:

  1. After Entering DB details, There will be a configuration page where you can give your site title and admin username and password
  2. Once you submit the form, You will redirect to wordpress admin login page.
  3. Login with mentioned username and password you have given in the form. That's It !!

Admin Settings

Main settings for your wordpress site can be done in wp-admin > settings page.

  1. Site Title & Tagline and URL settings in Settings > General
  2. Home page and posts settings in Settings > Reading
  3. Comments Settings in Settings > Discussion
  4. Post Images Settings in Settings > Media
  5. Post Links Settings in Settings > Permalinks

Themes - Folders & Basics

As per wordpress standard you are allowed to edit only the wp-content folder only, as this will helpful for the client can update the wordpress anytime.

in wp-content -> themes folder you can add/remove/modify your wordpress themes.

Go to appearance->themes page and select/activate your theme

wp-content > themes folder

By default, wordpress comes with three themes TwentyFourteen, TwentyFifteen and TwentySixteen

Copy & Paste one of the theme and rename it to your site title

Add/Replace screenshot.png in your theme folder.

goto wp-admin > appearance > themes page, you can see your screenshot there, clict to activate your theme

header.php - theme folder

This will be common header file for all your wordpress FO Pags. By default header.php will be a direct file in your theme folder

Add your favicon and logo in this file and check it in the browser.

Base path to include images, css & js: - Ex: http://localhost/exp/mySite/wp-content/themes/twentysixteen/
<?php echo get_template_directory_uri(); ?>

Base path for the menu: - Ex: http://localhost/exp/mySite/
<?php echo home_url(); ?>

functions.php - theme folder

All your theme settings(DEV Oriented) available here like image thumbnails, external file links, menu locations, widgets, etc.,

You can link css and js files here as below.

For CSS:
wp_enqueue_style( $handle, $src, $deps, $ver, $media = 'all' ); Ex: wp_enqueue_style( 'form', get_template_directory_uri().'css/form.css', array(), false );

For JS:
wp_enqueue_script( $handle, $src, $deps, $ver, $in_footer ); Ex: wp_enqueue_script( 'form', get_template_directory_uri().'js/form.js', ['jQuery'], '1.2', true );

style.css - theme folder

This will be the common stylesheet file for any of your wordpress theme. By default style.css will be a direct file in your theme folder

No need to include this style, as it was already included in functions.php

js/functions.js - theme folder

This will be the common script file for any of your wordpress theme. By default there will be mobile menu toggle functions, comment functions and theme related functions available.

No need to include this js, as it was already included in functions.php

frontpage.php - theme folder

This will be your custom home page. You have add to this in your theme folder, if it's not there.

Menu - Admin

You can add your menu in wp-admin > appearnce > menu.

  1. Give a menu name in add menu input box and click on the add menu button
  2. Add links from left sidebar
  3. You can change the positions of the menu link by dragging on it
  4. You can give your own class to your menu link in Screen Options Menu (placed in top right)

Menu - Code

In your file you can call the below code. It will generate links in << ul > li > a >> layout

<?php
  wp_nav_menu( array(
    'theme_location' => 'primary',
    'menu_class' => 'mainMenu',
  ));
?>

Links - Code

As mentioned earlier you can use <?php echo base_url; > to print site url and you can add your inline link using this.

Editors Basics

There is an auto generated <p> & <br/> issue in wordpress editor. To avoid this:

  1. Keep your texts in block elements like div, p, h1-h6. Don't add new line for inline elements like img, a, span.,
  2. Don't wrap a text line yourself unless you nedd a break tag there.
  3. Don't add comments in editor. Then your comments also generated with <p> tag.
  4. When you insert an image, make sure it was inside a figure tag or any block elements. Don't add images in the new line. Remove width and height from the img element.

Posts

You can add your pages in wp-admin > posts > add new.

  1. Give the post title in the title inputbox. It represents post title as well as the post link and also in any post lists
  2. Add some content in description box and publish the post.
  3. You can set the post feature image in Featured Image sidebar
  4. You can get the post link under title input box
  5. You can add categories and tags in respective sidebars.

Posts - Code

  1. By default page template will be in the theme folder > single-post.php
  2. the_post_thumbnail() - represents fetaured image of the post
  3. the_title() - represents the title of the post
  4. the_content() - represents the description/content of the post

Pages

You can add your pages in wp-admin > pages > add new.

  1. Give the page title in the title inputbox. It represents page title as well as page link and some times as page template too :)
  2. Add some content in description box and publish the page.
  3. You can get the page link under title input box
  4. You can page template in Parent select box in the Page Attributes sidebar.

Pages - Code

  1. By default page template will be in the theme folder > page.php
  2. You can create a new page like page-contact.php and create a page in the back office with the title contact. Then the page will take page-contact.php only
  3. You can create any page-name.php and give << //Template Name: Plain Text >> in the first line of the page. Then you can select this page template in admin
